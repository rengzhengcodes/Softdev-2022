Callouts

These are messages that are distinct in color from their surrounding contents but subdued to not clas with themes.

Secondary Callout

It is done by a div with the callout class, as well as one of the classes "secondary", "primary", "success", "warning", and "alert", which just change the background color and indicate the purpose of the callout
Adding a size class like "small" or "large" will change the size of the callout, adding more or less padding to it.

Horizontal Splitting

The basis of horizontal spacing in Foundation is the "grid-x" div class. This class is always split into a width of 12 cells wide.
Cells are integer widths, and are any container with the class "{size}-{span}" with size referring to the screen size (small, medium, or large) and span referring to how many cells horizontally it spans. They only work in the context of a grid-x div. Here are some examples of cell usage below.
