Horizontal Splitting

The basis of horizontal spacing in Foundation is the "grid-x" div class. This class is always split into a width of 12 cells wide.
Cells are integer widths, and are any container with the class "{size}-{span}" with size referring to the screen size (small, medium, or large) and span referring to how many cells horizontally it spans. They only work in the context of a grid-x div. Here are some examples of cell usage below.

Margins

Does what it says on the tin, adds margins to cells. You just make the grid-x div also a "grid-margin-x" div (append it to the end of the class name with a space, e.g. class="grid-x grid-margin-x")
